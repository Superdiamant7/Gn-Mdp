This password generator can generate strings of 30 random characters for a better security 🔐  
It uses customTkinter for the interface 💻  
  
Features:  
  
  - You can copy the password 😎  
  - You can choose the number of characters 😱  
  - It's free 🤑
  - And it's french 🟦⚪🟥
